My *taliored* post-install script for debian

	* Updates packages 
	* Full-upgrade

* Installations 
	* Gnome software and it's flatpak plugin
	* The inter font and reloads the font cache
	* The nvidia-driver
	* Flatpak
		* Adds the Flathub remote
		* Github desktop for linux
